# Twitter Sentiment Analysis

## Getting Started

### Install ntlk

``` 
pip install nltk

python3 

>>> import nltk 

>>> nltk.download() 
```

### Running

#### Knn
``` python3 knn.py ```

#### Svm
``` python3 svm.py ```

#### Round Robin
``` python3 rr.py ```

### Open Jupyter
``` jupyter notebook ```
